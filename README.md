# Crafto Light Bot

Мод для Minecraft, добавляющий умного бота, который может помогать игроку в различных задачах: защите от мобов, добыче ресурсов и строительстве.

## Установка

1. Убедитесь, что у вас установлен Minecraft Forge версии 47.1.0 для Minecraft 1.20.1. Если нет, скачайте и установите его с официального сайта Forge.

2. Скачайте jar файл мода `craftolight-1.0.0.jar` из релизов проекта.

3. Поместите jar файл в папку модов вашего Minecraft:
   - На Windows: `%APPDATA%\.minecraft\mods`
   - На macOS: `~/Library/Application Support/minecraft/mods`
   - На Linux: `~/.minecraft/mods`

4. Запустите Minecraft и выберите профиль с Forge 1.20.1-47.1.0.

5. В игре мод должен появиться в списке модов.

## Возможности бота

Бот представляет собой сущность, которая может выполнять следующие задачи:

- **Защита игрока**: Бот автоматически атакует ближайших враждебных мобов, помогая защищать игрока.
- **Добыча ресурсов**: Бот может искать и добывать алмазы в радиусе вокруг себя.
- **Строительство**: Бот может строить простые постройки, такие как дом из дубовых досок.

Бот привязан к игроку, который его вызвал, и может выполнять только одну задачу за раз.

## Команды

Все команды начинаются с `/bot`. Используйте их в чате игры.

- `/bot spawn` - Вызывает бота рядом с вами. Бот появится в вашей позиции.
- `/bot mine diamond` - Приказывает боту искать и добывать камень. Бот будет добывать до 10 блоков камня в радиусе 5 блоков.
- `/bot build house` - Приказывает боту построить небольшой одноэтажный дом с кроватью, освещением, крышей и сундуком внутри.
- `/bot followme` - Приказывает боту следовать за вами.

Бот также понимает команды на русском языке в чате (без /):

- "найди 10 алмазов" или "добыть алмазы" - бот начнет добывать камень.
- "построй дом" или "строить дом" - бот построит дом.

## Примечания

- Бот использует официальные маппинги Minecraft, совместим только с версией 1.20.1.
- Мод разработан на Java 21, но совместим с Forge.
- Если у вас возникли проблемы, проверьте логи Minecraft на ошибки.

## Разработка

Мод разработан с использованием Minecraft Forge MDK. Для сборки используйте `./gradlew build`.
